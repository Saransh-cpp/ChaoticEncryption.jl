ChaoticEncryption._redify — Method
_redify(colour)

Returns the red component of the RGB colour. Specifically developed to return an Array of r values of all the pixels of an image.

Running on an entire image - r_vals = _redify.(img)

Arguments

  • colour::RGB: Colour value with r, g, and b components.

Returns

  • r::Float64: r component of the RGB colour.

ChaoticEncryption._imageify — Method
_imageify(colour)

Returns the RGB type constructed with the provided R, G, and B values. Specifically developed to return the image constructed with the provided R, G, and B Arrays

Running on entire R, G, and B values - img = _imageify.(r, g, b)

Arguments

  • r::Float64: r component.
  • g::Float64: g components.
  • b::Float64: b components.

Returns

  • rgb::RGB: The RGB value.
